# Описание сценариев использования приложения
## Регистрация в приложении
Цель: создать учётную запись для доступа ко всем функциям приложения.
Участники: новый пользователь.
Предусловия: приложение установлено на устройство.

Шаги:

Запуск приложения, нажатие кнопки «Зарегистрироваться».

Выбор способа регистрации:

через email (example@mail.com);

через соцсети (Google, Apple ID, Facebook);

анонимный вход (с ограничением функционала).

Ввод персональных данных:

имя и фамилия;

дата рождения;

пол;

страна/регион.

Указание спортивных интересов (выбор из списка: бег, йога, велоспорт и т.п.).

Установка цели (похудение, подготовка к соревнованию, поддержание формы).

Создание пароля (не менее 8 символов, с цифрами и спецсимволами).

Подтверждение email/номера телефона через код из SMS или письма.

Прохождение краткого онбординга (обзор ключевых функций).

Переход в личный профиль.

Постусловия: пользователь получает доступ к полному функционалу.
Альтернативные пути:

Ошибка ввода email → сообщение «Некорректный формат email».

Слабый пароль → подсказка «Добавьте цифры и символы».

2. Проведение тренировки
Цель: выполнить тренировку с отслеживанием показателей.
Участники: зарегистрированный пользователь.

Шаги:

Открытие раздела «Тренировки».

Выбор типа активности:

из рекомендованного плана;

вручную (бег, плавание, силовые и т.п.).

Настройка параметров:

длительность;

интенсивность;

цель (сжигание калорий, выносливость).

Нажатие «Начать тренировку».

Во время занятия:

отслеживание пульса (если подключено устройство);

фиксация пройденного расстояния;

паузы через кнопку «Перерыв».

Завершение тренировки → нажатие «Закончить».

Просмотр отчёта:

потраченные калории;

средняя скорость;

график пульса.

Добавление заметок («Чувствовал усталость на 3‑й км»).

Публикация результата в социальной ленте (опционально).

Постусловия: данные сохраняются в истории тренировок.
Альтернативные пути:

Сбой GPS → сообщение «Проверьте подключение к сети».

Низкий заряд устройства → предупреждение «Сохраните данные, чтобы не потерять прогресс».

3. Сравнительный анализ тренировок
Цель: оценить прогресс и сравнить результаты.
Участники: пользователь с историей тренировок.

Шаги:

Переход в раздел «Аналитика».

Выбор периода сравнения:

неделя/месяц/год;

конкретные даты.

Выбор параметров для сравнения:

дистанция;

время;

пульс;

калории.

Активация режима сравнения:

«Со мной прошлым месяцем»;

«С пользователями моего уровня»;

«С профессионалами».

Просмотр визуализаций:

графики динамики;

таблицы с цифрами;

цветовые индикаторы улучшений (зелёный/красный).

Получение рекомендаций:

«Ваша выносливость выросла на 15%!»;

«Попробуйте увеличить нагрузку на 10%».

Постусловия: пользователь видит объективную оценку прогресса.
Альтернативные пути:

Нет данных для сравнения → сообщение «Начните тренировки, чтобы увидеть динамику».

4. Найти друга и отправить запрос
Цель: расширить социальный круг в приложении.
Участники: авторизованный пользователь.

Шаги:

Открытие раздела «Сообщество».

Переход во вкладку «Найти друзей».

Использование фильтров:

местоположение (радиус от 1 до 50 км);

виды спорта;

уровень подготовки.

Просмотр профилей потенциальных друзей:

фото;

статистика тренировок;

интересы.

Нажатие кнопки «Добавить в друзья» у выбранного профиля.

Ввод персонального сообщения (опционально):
«Привет! Давай тренироваться вместе!»

Ожидание подтверждения запроса.

При принятии запроса — автоматическое добавление в список друзей и открытие чата.

Постусловия: пользователи могут общаться и видеть активность друг друга.
Альтернативные пути:

Пользователь заблокирован → сообщение «Невозможно отправить запрос».

Профиль скрыт → фильтр не показывает такого пользователя.

5. Найти и вступить в группу
Цель: присоединиться к сообществу по интересам.
Участники: зарегистрированный пользователь.

Шаги:

Открытие раздела «Группы».

Использование поиска:

по названию («Бегуны Москвы»);

по хештегам (#йога_для_начинающих);

по тематике.

Просмотр информации о группе:

описание;

количество участников;

последние посты.

Проверка типа группы:

открытая (вступление без одобрения);

закрытая (требуется одобрение администратора).

Нажатие «Вступить в группу».

Для закрытых групп — ожидание подтверждения от модератора.

После вступления:

подписка на уведомления;

доступ к чату и файлам группы;

возможность публиковать посты.

Постусловия: пользователь становится членом сообщества.
Альтернативные пути:

Группа не найдена → предложение создать свою.

Отказ в вступлении → сообщение «Группа закрыта для новых участников».

6. Подключение устройств
Цель: синхронизировать внешние датчики с приложением.
Участники: пользователь с совместимым устройством.

Шаги:

Переход в раздел «Настройки» → «Подключённые устройства».

Нажатие «Добавить устройство».

Выбор типа устройства:

смарт‑часы;

пульсометр;

шагомер;

датчик кислорода.

Включение Bluetooth на телефоне и устройстве.

Поиск доступных устройств в списке.

Выбор нужного гаджета → ввод PIN‑кода (если требуется).

Проверка синхронизации данных:

тестовое измерение пульса;

отображение шагов.

Настройка параметров сбора данных:

частота обновлений;

уведомления о аномалиях.

Сохранение настроек.

Постусловия: устройство передаёт данные в приложение.
Альтернативные пути:

Устройство не найдено → проверка заряда и режима сопряжения.

Ошибка синхронизации → предложение перезагрузить устройство.

Несовместимость → сообщение «Это устройство не поддерживается».
